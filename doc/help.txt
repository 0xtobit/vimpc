Vimpc Help
==================================================

NORMAL MODE
--------------------------------------------------
   [count] .                      | repeat the last normal mode action, if count is specified
                                  | changes the count that was used for the last action to [count]

MOTION:
   [count] k, Up                  | scroll up in current window

   [count] j, Down                | scroll down in current window

   [count] PageUp, ctrl+u         | scroll up by a page in current window

   [count] PageDown, ctrl+d       | scroll down by a page in current window

           Home                   | scroll to top of the screen
           
           End                    | scroll to the bottom of the screen
   
   [count] f                      | scroll to the currently playing song
                                  | if [count], then scrolls to current song + [count]

   [count] F                      | same as f, only [count] songs before the current
   
   [count] gg                     | goes to top of the screen, or the line [count] if specified
   [count] G                      | goes to bottom of the screen, or the line [count] if specified

   [count] H                      | selects the song at the top of the currently active section of window,
                                  | if count is specified, selects [count] down from the top.
   [count] M                      | selects the song in the middle of the currently active section of window.

   [count] L                      | selects the song at the bottom of the currently active section of window,
                                  | if count is specified, selects [count] up from the bottom.

   [count] z<Enter>               | moves the currently selected line to the top
                                  | if count is specified, moves the [count]th line to the top.

   [count] z.                     | centres the currently selected line
                                  | if count is specified, centres the [count]th line.

   [count] z-                     | moves the currently selected line to the bottom
                                  | if count is specified, movies the [count]th line to the bottom.

WINDOWS/TABS:
   [count] gt                     | change to the next window, or to the [count] window if count is specified
   [count] gT                     | change to the previous window, or change back [count] windows if count is specified
           Alt+<window number>    | change to the given window number
                                 
SEARCHING:                        
           /<exp>                 | searches forward for <exp> (a regular expression)
           ?<exp>                 | searches backward for <exp> (a regular expression)
   [count] n                      | go to the next result
   [count] N                      | go to the previous result

SEARCH FLAGS:
          \c                      | when used in search pattern, disables case sensitivity

PLAYING (ALSO SEE PLAYLIST):                          
   [count] <                      | skip to the previous song, if count is specified, skips back [count] songs
   [count] >                      | skip to the next song, if count is specified, skips forward [count] songs
   [count] [                      | skip to the previous artist, if count is specified, skips back [count] artists
   [count] ]                      | skip to the next artist, if count is specified, skips forward [count] artists
   [count] {                      | skip to the previous album, if count is specified, skips back [count] albums
   [count] }                      | skip to the next album, if count is specified, skips forward [count] albums

   [count] +                      | increase volume by 1 or [count]
   [count] -                      | decrease volume by 1 or [count]

           p                      | toggle the pause on/off
           s                      | stop playback

           C                      | toggles consume
           R                      | toggles random
           E                      | toggles repeat
           S                      | toggles single


ALL WINDOWS:
           A                      | add all songs to the playlist
           D                      | delete all songs from the playlist

PLAYLIST:
   [count] d                      | delete the next [count] songs from the playlist
           P                      | paste the previously deleted songs above the current selection
           <Enter>                | play the currently selected song

BROWSE:
   [count] a                      | add the next [count] songs
   [count] d                      | delete the next [count] songs, if they are in the playlist
           <Enter>                | append the selected song to the playlist and begin playing it

LISTS:
   [count] a                      | add the songs from the next [count] playlists to the queue
   [count] d                      | delete the next [count] playlists
           <Enter>                | play the currently selected playlist
   

LIBRARY:
           a                      | add the currently selected song, album or artist to the current playlist
           h, Left                | collapse current selection into parent
           l, Right               | expand the current selection
           <Enter>                | clear the current playlist, add the currently selected song/album/artist and begin playing


COMMAND/EX MODE
--------------------------------------------------

          :<command>              | executes the command <command>
                                  | to execute a command the full-name does not need to be given provided that
                                  | it is a unique match ie :con does not work as it could be :console or :connect
                                  | but :conn will call the :connect command

PLAYBACK:
          next [count]            | skips forward to the next song, or if given the next [count] songs
          previous [count]        | skips backward to the previous song, or if given the previous [count] songs
          play <song id>          | play the song with the given id in the current playlist
          stop                    | stop playback

          volume [0-100]          | set the volume

          random [on/off]         | toggles random on/off
          repeat [on/off]         | toggles repeat on/off
          single [on/off]         | toggles single on/off
          consume [on/off]        | toggles consume on/off

          shuffle                 | shuffle the playlist

PLAYLISTS:
          load <name>             | load the playlist called <name>
          edit <name>             | same as load
          save <name>             | save the current playlist as <name>
          write <name>            | same as save
                                  

WINDOWS/TABS:
          browse                  | change to the browse window, if window is hidden it is opened
          console                 | change to the console window, if window is hidden it is opened
          help                    | change to the help window (this one), if window is hidden it is opened
          library                 | change to the library window, if window is hidden it is opened
          playlist                | change to the playlist window, if window is hidden it is opened

          tabfirst                | navigate to the first tab
          tablast                 | navigate to the last tab
          tabmove <location>      | move the current tab to new location (first location is 0)
          tabhide [tabname]       | hide the current tab (or the one specified) from the tab bar, will change active window if necessary

MISC:
          !mpc <command-string>   | executes the command line program mpc with the given <command-string>
                                  | the result is output in the console window

          alias <name> <commands> | creates an alias called <name> which executes the given commands,
                                  | each command may be seperated by ;
                                  | i.e alias hello echo "hello"; echo "world";

          clear                   | clears the playlist window

          connect [host] [port]   | connects to the given hostname if none is specficied, uses localhost
                                  
          echo <string>           | echos the given string to the console window
                                  
                                  
          quit[!]                 | quit vimpc, if [!] is appended playback will be stopped regardless of stoponquit setting
                                  
          redraw                  | redraws the current window
                                  
          set                     | sets a configurable setting, see SETTINGS section
                                  

SETTINGS
--------------------------------------------------

         All settings can be prefixed with no, which will disable the setting, or postfixed with ! which will toggle the
         setting.

            [no]autoscroll[!]     | automatically scroll to the playing song, when a h, l, q, w, Q, W, etc is performed 

            [no]colour[!]         | enable or disable colours

            [no]expand-artists[!] | when enabled will expand to the artist level by default in the library window
                                  | if you change this during operation you will need to call :redraw from the library window
                                  
            [no]hlsearch[!]       | highlight search results

            [no]ignorecase[!]     | case insensitive searching
                                  
            [no]searchwrap[!]     | wrap search results when using n/N
                                  
            [no]stoponquit[!]     | automatically stop playback when vimpc is quit

            [no]timeremaining[!]  | display the time that is left in the song rather than elapsed
                                  
            [no]windownumbers[!]  | display the number next to the window in the tab bar
                                  
            window <window>       | sets the window that will start with focus, only useful if called from ~/.vimpcrc file
